============================================================
frontend/src - 前端源代码目录
============================================================

【文件夹作用】
Vue 3应用的核心源代码。

【子文件夹/文件说明】
- api/           API请求封装层
- assets/        静态资源（图片、样式、地图数据）
- components/    所有Vue组件（按功能组织）
- composables/   可复用组合式函数
- stores/        Pinia状态管理
- App.vue        根组件
- main.js        应用入口，挂载Vue实例
- env.d.ts       TypeScript环境声明
